 Data Cleaning for State-Level Cigarette Sales (1970-2000)

- Run data_clean_cig_all_state.m to clean the raw Cigarette data. 

- Download the raw dataset from: 
    https://chronicdata.cdc.gov/Policy/The-Tax-Burden-on-Tobacco-1970-2019/7nwe-3aj9/data

- Raw dataset: The_Tax_Burden_on_Tobacco__1970-2019.csv

- Keep Variables: Year, State abbreviation, State name, and per capita consumption

- Time Range: 1970-2000. 

- Sort by Year and State abbr.

- Save as cigs_all_state.xls
